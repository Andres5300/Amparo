# Amparo Landing Page

Landing page de alta conversión para Amparo, construida con React + Tailwind CSS.

## Requisitos

- Node.js 18+
- npm 9+

## Ejecutar en local

1. Instala dependencias:
   npm install
2. Inicia el entorno de desarrollo:
   npm run dev
3. Abre la URL que muestra Vite en consola.

## Build de producción

npm run build
npm run preview

## Estructura principal

- src/App.jsx
- src/components/
- src/constants/links.js
- src/index.css
